# ☁️ Cloud Data Redundancy Removal System

A cloud-based web application that detects and removes duplicate user data using Flask and MongoDB Atlas.

---

## 🚀 Features

- User data storage
- Duplicate email detection
- Data redundancy prevention
- MongoDB Atlas cloud database
- Delete user functionality
- Professional dashboard UI
- Cloud deployment using Render

---

## 🛠 Technologies Used

- Python
- Flask
- MongoDB Atlas
- HTML/CSS
- Render
- GitHub

---

## ☁️ Cloud Computing Concepts

- Cloud-hosted database
- Real-time cloud CRUD operations
- Multi-user access
- Cloud deployment
- Scalable architecture

pip install -r requirements.txt

### Run project

python app.py
